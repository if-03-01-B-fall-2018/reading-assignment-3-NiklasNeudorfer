HTML for Lists

Start Tags for:

  • an ordered list: < ol>
  • an unordered list: < ul>
  • a description list: < dl>
  • a list item: < li>
  • a description term: < dt>
  • a description data: < dd>

CSS for Lists

  1. Make bullet a square:
    • ul.b { list-style-type: square;}
  2. Make bullet an image named "my-bullet.png":
    • ul {list-style-image: url('my-bullet.png');}
  3. Make bullet an uppercase roman number:
    • ul.b { list-style-type: upper-roman;}
  4. Set bullet positionto be inside the list item's text flow:
    • ul.b { list-style-position: inside;}
  5. Set bullet position to be outside the list item's text flow:
    • ul.b { list-style-position: outside;}

HTML for Tables

Start Tags for:

  • a table: < table>
  • a table row: < tr>
  • a table header: < thead>
  • a table header cell: < th>
  • a table body: < tbody>
  • a table cell: < td>
  • a table footer: < tfoot>
  • a table footer cell: < td>

CSS for Tables

  1. Black dotted border of 1px width
    • th, td { border: 1px solid black;}
  2. Solid border of 1 px width having a red color and rounded corners
    • table th, td { border: 1px solid red;border-radius 6px; }
  3. Merge the borders of table cells so that they appear as one line
    • th, td { border-bottom: 1px solid#ddd;}
